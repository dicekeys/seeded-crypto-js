Interface SignatureVerificationKey

A SignatureVerificationKey is used to verify that messages were signed by its corresponding SigningKey. SigningKeys generate _signatures_, and by verifying a message/signature pair the SignatureVerificationKey can confirm that the message was indeed signed using the SigningKey. The key pair of the SigningKey and SignatureVerificationKey is generated from a seed and set of derivation specified options in specified using the JSON Derivation Options format.

To derive a SignatureVerificationKey from a seed, first derive the corresponding SigningKey and then call SigningKey.getSignatureVerificationKey.

The static methods of this class are documented in SigningKeyStatic.

For code examples, see SigningKey.

You must delete this object manually by calling delete to avoid memory leaks, as it is stored within the WebAssembly module and is not automatically garbage collected by the JavaScript runtime.
